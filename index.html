Featured Project

PROJECT PRD & PLAN

🎬 Beyond the Stream: Netflix’s Global Success Formula in 2025

Overview: 2025년 최신 넷플릭스 데이터를 활용하여 글로벌 OTT 시장 내 콘텐츠의 상업적 흥행(Popularity)과 작품성(Rating) 사이의 상관관계를 탐구하는 데이터 분석 웹 대시보드입니다.

💡 Project Plan (5W1H)

  • Why: 넷플릭스가 취하고 있는 콘텐츠 로컬라이제이션 전략의 실체를 상업성 지표와 비평적 가치 대조를 통해 논리적으로 증명하기 위함.
  • Who: 데이터 기반의 OTT 콘텐츠 트렌드 분석 및 글로벌 마케팅 전략 수립에 관심 있는 기획자 및 평가단.
  • What: 영화/TV쇼 데이터 통합, 인기도 vs 평점 산점도 분석, 글로벌 언어권별 제작 비중 추적, 실시간 작품 상세 검색.
  • When: 성균관대학교 기말 프로젝트(W13-W16) 발표 현장 및 데이터 기반 콘텐츠 제작 시나리오 기획 시 활용.
  • Where: GitHub 저장소 호스팅 및 Streamlit Cloud 기반의 웹 브라우저 환경.
  • How: Python 언어를 기반으로 Pandas, Streamlit, Plotly 라이브러리를 결합하여 구현.

🛠️ 3-Step Development Roadmap

Step 1: Framework 대용량 영화 및 TV 쇼 상세 CSV 데이터를 Pandas로 결합하고, 핵심 KPI 지표를 도출하는 기본 구조 설계.
Step 2: Interactivity 다차원 사이드바 필터(장르, 연도, 평점)를 추가하고 Plotly를 연동하여 동적 시각화 구현.
Step 3: Polish 실시간 텍스트 기반 작품 검색 기능을 고도화하고, Streamlit Cloud 배포 및 최종 문서화 완료.
